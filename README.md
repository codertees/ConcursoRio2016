INSTRUCCIONES DE USO:

Para mayor rapidez, se cargan los datos del ejemplo puesto en la pagina del concurso pero estos se pueden modificar dando click derecho sobre la fila del país que se quiera eliminar y escoger la opción ELIMINAR PAÍS. De esta misma manera también se pueden modificar cada una de las filas (países) ingresados. Si no hay ningún país seleccionado NO SE PUEDE ELIMINAR NI MODIFICAR.

Para agregar PAÍSES, llenamos los datos de NOMBRE, CANTIDAD DE MEDALLAS DE ORO, CANTIDAD DE MEDALLAS DE PLATA y CANTIDAD DE MEDALLAS DE BRONCE. Si falta alguno de estos datos no podemos añadirlo. Si están completos presionamos ENTER o el botón + (AGREGAR) y se añadirá a la lista SIEMPRE Y CUANDO EL PAÍS NO ESTE AGREGADO.

Para ordenar la lista, presionamos el botón ORDENAR, que esta al lado del botón + (AGREGAR). Primero se verifican por cantidad de medallas de bronce, luego por las de plata y por ultimo por las de oro, es decir de menor a mayor valor para organizar correctamente

Necesitan tener .NET FRAMEWORK 4.5 para ejecutar.
